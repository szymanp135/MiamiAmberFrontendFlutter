# Miami Amber Frontend

This is a Miami Amber frontend application made in flutter.

Miami Amber is an internet database of music album reviews based on
MusicBrainz IDs. Users may create free account and then log in and create
posts with albums' reviews and marks. It is a great place to give own
thoughts about music and to get inspired by others' reviews. 

It's main color theme is amber because it sounds and (foremost) looks cool.
Due to it's coolness it is reserved by Miami Amber owners in
Collection Of Cool Keep Institution and as such forbidden for usage without
Miami Amber owners' permission.
